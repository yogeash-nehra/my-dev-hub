### Claude Agent SDK Python v0.1.73 — Eager Session Store Flushing
**Source:** anthropics/claude-agent-sdk-python GitHub | **Date:** 2026-05-04 | **Link:** https://github.com/anthropics/claude-agent-sdk-python/releases/tag/v0.1.73
**TL;DR:** New `session_store_flush` option (`"batched"` | `"eager"`) on `ClaudeAgentOptions` delivers session frames to `SessionStore.append()` in near-real-time instead of only at end-of-turn.
**Dev signal:** `eager` mode enables three previously difficult patterns: (1) live-tailing UIs that stream agent progress, (2) cross-process agent resume from another worker, (3) crash-durability — agent state is already persisted before a crash. Set `session_store_flush="eager"` if you're building any of these. Default remains `"batched"`. Run `pip install claude-agent-sdk==0.1.73`.
**Primary source:** https://github.com/anthropics/claude-agent-sdk-python/releases/tag/v0.1.73
